# Level 3 Draft Proposals

Files in this directory are public Level 3 draft proposals for review and discussion.

They are not canonical AI Atlas taxonomy.

The canonical Level 1 and Level 2 taxonomy remains in:

- `taxonomy/ai-taxonomy-l1-l2.json`
- `taxonomy/ai-taxonomy-l1-l2.md`

Promotion of any Level 3 draft requires a separate, explicit review and approval decision. Adding or updating a file in this directory does not promote its contents.

Generated public views are not updated from these draft files.

Draft proposals should avoid products, company names, concrete model versions, and temporary vendor labels unless they are explicitly in scope for a later review.
