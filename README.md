# AI Atlas

A clean, open map of modern Artificial Intelligence.

## Mission

To build an open, community-reviewed map of modern AI that helps people understand how the field fits together.

## Current status

v0.1 — Clean hierarchy draft, Level 1 and Level 2 only.

## What this is

AI Atlas is an open AI taxonomy project.

It is not intended to be an authoritative taxonomy yet. The goal is to create a useful, discussable, expandable mental model of AI that can be reviewed and improved by the community.

The project starts with a clean hierarchy before expanding into detailed concepts, model families, architectures, systems, and products.

For now, the canonical source is the Level 1 and Level 2 taxonomy.

## Typed knowledge map

AI Atlas is not only a tree.

Each concept has:

- a hierarchy level,
- a hierarchy level name,
- a concept type,
- stability metadata.

This allows the project to distinguish between:

- fields,
- major areas,
- subfields,
- paradigms,
- methods,
- architectures,
- model families,
- techniques,
- systems,
- products,
- safety concepts,
- evaluation concepts,
- future concepts.

This matters because AI concepts do not all belong to the same kind of category.
